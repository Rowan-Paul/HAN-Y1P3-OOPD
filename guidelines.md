# Guidelines
If you wish to contribute to this repo, please follow the following guidelines.

## Language
The project itself is written in Java with classes, functions, tabs, comments, etc being named in English. The game itself has a user-facing interface in Dutch. In principle the entire project code should be readable in English.

## Features
Planned features are being tracked on the [Github Projects](https://github.com/Rowan-Paul/ICT-Y1P3-OOPD/projects) page which will be regularly updated to reflect the current state of the project.
