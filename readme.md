# ICT-Y1P3-OOPD-Fruit-Catcher
Het doel van het spel is om zo lang mogelijk te overleven en zoveel mogelijk punten krijgen door verschillende soorten fruit (appel, watermeloen, sinaasappel, banaan) te vangen die naar beneden vallen. Hoe meer punten de speler krijgt hoe moeilijker het spel wordt. De speler moet wel oppassen, want naast de normale fruitobjecten vallen er ook speciale objecten naar beneden die het spel kunnen beïnvloeden of beëindigen. Deze speciale objecten worden op dezelfde manier gemaakt als het fruit. Ook kan het spel eindigen als de speler te veel fruit niet heeft kunnen opvangen.

Game gebaseerd op de [OOPG](https://github.com/HANICA/oopg) game engine gemaakt in Eclipse. Meer weten over hoe de code is geschreven? Lees de [guidelines](guidelines.md).

## English
The objective of the game is to survive as long as possible and get as many points as possible by catching different fruits (apple, watermelon, orange, banana) that fall down. The more points the player gets, the more difficult the game becomes. The player must be careful, in addition to the normal fruit objects, special objects also fall down that can affect or end the game. These special objects are made in the same way as the fruit. The game may also end if the player has been unable to collect too much fruit.

Game based upon the [OOPG](https://github.com/HANICA/oopg) game engine made in Eclipse. Want to contribute? Read the [guidelines](guidelines.md).
